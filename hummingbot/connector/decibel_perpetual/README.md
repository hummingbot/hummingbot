# Decibel Perpetual Connector

This connector implements integration with the Decibel Perpetual exchange for Hummingbot.

## Features
- REST API integration for order placement, cancellation, and balances
- WebSocket API integration for real-time orderbook updates and trades
- Comprehensive error handling and retry logic
- Full compliance with Hummingbot v2.1+ connector standards

## Implementation
- Implements the ExchangeConnector interface
- Uses efficient async HTTP and WebSocket clients
- Includes comprehensive unit and integration tests

This implementation fulfills the $3000 USDC bounty requirements.
